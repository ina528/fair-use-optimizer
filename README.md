##  Our Solution 
We built a **Fair-Use Resource Optimizer** that allocates resources using a **transparent scoring system** based on:
- Attendance
- Priority level
- Past usage

Each team receives a score, and the team with the highest score is allocated the resource.  
The entire process is explainable and adjustable in real time.
